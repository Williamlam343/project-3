# Soles 'R Us

## Description
Soles 'R Us is a friendly ecommerce app created to help users find the latest shoes from the hottest brands at competitive prices. Users are also able to add shoes to a cart and checkout.

## Technologies Used
* React
* MongoDB/Mongoose
* Materilize
* Stripe
* Apollo
* GraphQl
